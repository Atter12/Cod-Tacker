# Roadmap

This document describes planned work only. The following integrations and workers are not implemented by the current admin panel.

## Next phases

1. Shopify ingestion and operational sync status.
2. Meta Ads account, campaign, and spend ingestion.
3. TikTok Ads ingestion.
4. WhatsApp conversation and message ingestion.
5. Carrier connections, tracking events, and status normalization.
6. Reconciliation ingestion and settlement matching.
7. Background workers, retry policies, dead-letter processing, and operational observability.

Each phase should include least-privilege credentials, idempotent ingestion, RLS coverage, audit events, and explicit operational runbooks before appearing as implemented product functionality.
